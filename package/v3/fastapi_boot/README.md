# fastapi_boot

FastAPI项目启动器

特点：
- 类视图，CBD(class based view)
- 控制反转，IOC(Inversion of Controll)
- 依赖注入，DI(Dependency Injection)
- 自动扫描目录
- 待探索...

APIs：

- 控制器相关装饰器：
   1. `Controller`、`Prefix`
   2. `Get`、`Post`、`Socket`...
- 依赖注入
   1. `Injectable`(`Service`、`Repository`、`Component`)、`Bean`
   2. `Inject`(`Autowired`)
- 有用的`hook`
   1. `usedep`
   2. 待扩展
